### 2. 数据库配置 (application.yml)
```yaml
spring:
  datasource:
    # H2 数据库配置（开发和测试使用）
    url: jdbc:h2:mem:testdb;DB_CLOSE_DELAY=-1;DB_CLOSE_ON_EXIT=FALSE;MODE=MySQL
    username: sa
    password: 
    driver-class-name: org.h2.Driver
    hikari:
      minimum-idle: 5
      maximum-pool-size: 100
    # 数据库初始化配置
    initialization-mode: always
    schema: classpath:schema.sql
    continue-on-error: false
  
  # H2 控制台配置（开发环境使用）
  h2:
    console:
      enabled: true
      path: /h2-console
      settings:
        web-allow-others: true

mybatis-plus:
  initialization-mode: always
```

1. **内存数据库特性**：H2 内存数据库在应用停止后数据会清空，下次启动会重新初始化
2. **开发环境使用**：此配置仅适用于开发和测试环境，生产环境请使用 MySQL
3. **自动初始化**：每次启动都会自动执行 schema.sql 创建表和插入测试数据
