```

├── backend/                    # 后端 Spring Boot 项目
│   ├── src/                   # 源代码
```

## 🚀 快速启动

### 1. 启动数据库
```bash
# 方式一：使用 Docker（推荐）
docker start mysql80

# 方式二：双击批处理文件
.\启动 MySQL-Docker.bat
```

### 2. 启动后端服务
```bash
cd backend
mvn clean spring-boot:run
```
- 访问地址：http://localhost:8080

### 3. 启动前端服务
```bash
cd frontend
npm run dev
```
- 访问地址：http://localhost:3000

## 📝 SQL 脚本说明

### sql-scripts/ 目录
- **create_*.sql** - 创建新表的脚本
- **fix_*.sql** - 修复表结构或数据的脚本
- **insert_*.sql** - 插入测试数据的脚本
- **add_*.sql** - 添加索引、字段等的脚本

### data-import/ 目录
- **import_*.sql/ps1/py** - 从 CSV/Excel 等文件导入数据的脚本
- **init_db.sql** - 完整的数据库初始化脚本（包含所有表结构和基础数据）
- **budget_import*.sql** - 预算数据专用导入脚本

## 🔧 开发工具

- **JDK**: 17+
- **Maven**: 3.9.6+

- **MySQL**: 8.0+
- **Docker**: 用于运行 MySQL

## 📌 注意事项

1. **数据库文件**：`mysql-data/` 目录包含实际的数据库文件，请勿手动修改
2. **SQL 脚本**：执行前请备份数据，建议先在测试环境验证
3. **文档归档**：`docs-archive/` 存放历史文档，新文档请放在 `docs/` 目录
4. **端口占用**：确保 8080（后端）、3000（前端）、3306（数据库）端口未被占用
