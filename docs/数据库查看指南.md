## 📋 常用查询示例

### 查看内控预警数据
```sql
SELECT 
    log_id,
    alert_type,
    LEFT(alert_content, 50) as content,
    handle_status,
    handler
FROM internal_control_log 
ORDER BY alert_time DESC 
LIMIT 10;
```

### 查看用户列表
```sql
SELECT id, username, real_name, role FROM user;
```

### 统计各表记录数
```sql
SELECT 'internal_control_log' AS 表名, COUNT(*) AS 记录数 FROM internal_control_log
UNION ALL
SELECT 'user', COUNT(*) FROM user
UNION ALL
SELECT 'purchase_order', COUNT(*) FROM purchase_order
UNION ALL
SELECT 'production_order', COUNT(*) FROM production_order
ORDER BY 记录数 DESC;
```

### 查看采购订单（最近10条）
```sql
SELECT order_no, supplier_name, total_amount, status, create_time 
FROM purchase_order 
ORDER BY create_time DESC 
LIMIT 10;
```

## 🔧 高级用法

### 导出表数据为 CSV
```powershell
docker exec mysql80 mysql -uroot -p123456 shuju --default-character-set=utf8mb4 -e "SELECT * FROM user;" | Out-File -FilePath "users.csv" -Encoding utf8
```

### 备份整个数据库
```powershell
docker exec mysql80 mysqldump -uroot -p123456 shuju --default-character-set=utf8mb4 > backup.sql
```

### 恢复数据库
```powershell
docker exec -i mysql80 mysql -uroot -p123456 shuju --default-character-set=utf8mb4 < backup.sql
```

### 查看数据库大小
```sql
SELECT 
    table_schema AS '数据库',
    ROUND(SUM(data_length + index_length) / 1024 / 1024, 2) AS '大小(MB)'
FROM information_schema.tables
WHERE table_schema = 'shuju'
GROUP BY table_schema;
```

## 💡 提示

1. **中文显示**: 始终使用 `--default-character-set=utf8mb4` 参数确保中文正确显示
2. **数据安全**: 生产环境操作前请先备份
3. **性能优化**: 大数据量查询时使用 LIMIT 限制返回行数
4. **字段名称**: 不确定字段名时，先用 `DESCRIBE 表名` 查看表结构

## 🆘 常见问题

### Q: 如何查看某个表的完整结构？
```powershell
docker exec mysql80 mysql -uroot -p123456 shuju --default-character-set=utf8mb4 -e "SHOW CREATE TABLE 表名;"
```

### Q: 如何查找包含特定关键词的记录？
```sql
SELECT * FROM internal_control_log WHERE alert_content LIKE '%预算%';
```

### Q: 如何统计某个月的数据？
```sql
SELECT COUNT(*) FROM purchase_order 
WHERE MONTH(create_time) = 3 AND YEAR(create_time) = 2025;
```
